# 分享图片说明

由于无法直接创建图片文件，请用户自行准备分享图片：

## 要求
- 文件名：share-image.png
- 路径：static/share-image.png
- 尺寸：建议 1200×960 像素（5:4 比例）
- 大小：不超过 5MB
- 格式：PNG 或 JPG

## 设计建议
1. 使用品牌主色调（#1E40AF 蓝色）
2. 包含小程序名称 "OrderEase"
3. 包含吸引人的标语，如"便捷点餐，美味生活"
4. 可以包含食物图片或点餐相关图标

## 示例内容
- 背景色：渐变蓝色（#1E40AF 到 #3B82F6）
- 中心文字：OrderEase Logo 或名称
- 底部文字：快来点餐吧
- 装饰元素：食物图标或餐具图标

## 创建步骤
1. 使用设计工具（如 Photoshop、Figma、Canva）创建图片
2. 导出为 PNG 格式，尺寸 1200×960 像素
3. 保存到项目的 static 目录，命名为 share-image.png
4. 确保图片大小不超过 5MB

## 测试
创建完成后，在微信开发者工具中测试分享功能：
1. 点击右上角菜单
2. 选择"转发给朋友"
3. 检查分享图片是否正确显示
